## Para executar siga as seguintes instruções:

- Clona o ropositório

  ```bash
  git clone https://github.com/lucas-silva-dev/rdstationteste.git
  ```

- Instalar as dependências do projeto

  - yarn

  ```bash
  yarn install
  ```

  - npm

  ```bash
  npm install
  ```

- Acessar a pasta do projeto

  ```bash
  cd rdstationteste
  ```

- Rodar o projeto

  - yarn

  ```bash
  yarn dev
  ```

  - npm

  ```bash
  npm run dev
  ```
